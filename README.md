# 罗峪村村民积分系统

湖南体育职业学院驻村工作队 × 中共桑植县凉水口镇罗峪村支部委员会

## 项目简介

面向农村基层治理的村民积分管理系统，支持积分录入、审核、排行、兑换、公告等功能。
结合罗峪村党建引领、体育特色、生态保护三大板块设计。

## 技术栈

- 前端：H5 单页应用
- 后端：Node.js + Express
- 数据库：MySQL 8.0
- 部署：阿里云 ECS + 宝塔面板

## 功能模块

- 村民端：积分查询、明细、组内排行、物资兑换、公告
- 管理端：积分录入（含图片）、审核、数据统计、公告发布
- 权限体系：超级管理员 / 村干部 / 组长 三级角色
